## Bước thực hiện

1. Đọc file context tổng quan dự án:
```
Đọc file: PROJECT_CONTEXT.md tại thư mục gốc dự án
```

2. Nếu cần thêm chi tiết về một phần cụ thể, đọc thêm:
- Entity: `src/main/java/com/fpt/entity/`
- Service: `src/main/java/com/fpt/service/`
- Controller: `src/main/java/com/fpt/controller/`
- Config: `src/main/java/com/fpt/config/`
- application.properties: `src/main/resources/application.properties`

## Thông tin nhanh

- **Dự án:** Hệ thống quản lý Quán Nước Hương Tre
- **Stack:** Java 17 + Spring Boot 2.7.18 + MySQL + JWT + WebSocket + PayOS
- **Port:** 8081
- **Swagger:** http://localhost:8081/swagger-ui/index.html
- **Base package:** com.fpt
